# 🏋️ 普拉提门店经营分析 Skill

自动导入数据、生成RFM会员分层、趋势分析、流失预警、可视化看板。

## 功能特性

| 功能 | 说明 |
|------|------|
| 📥 数据导入 | 批量导入Excel预约记录，自动识别门店 |
| 🗄️ SQLite仓库 | 自动建表，支持历史数据查询 |
| 👥 RFM分析 | 最近消费/消费频率/消费金额三维评分 |
| 🎯 会员分层 | 高价值/活跃/新会员/沉睡/流失 五分类 |
| ⚠️ 流失预警 | 按风险等级自动预警 |
| 📊 可视化看板 | ECharts交互式图表，直观展示 |

## 使用方法

### 1. 完整流程（推荐）
导入数据 + 分析 + 生成看板一条龙完成：

```bash
/claude pilates-analyzer --action full \
    --month 2026-05 \
    --data-path /path/to/excel/files
```

### 2. 单独执行某个步骤

**只导入数据**：
```bash
/claude pilates-analyzer --action import --data-path /path/to/excel
```

**只执行分析**：
```bash
/claude pilates-analyzer --action analyze --month 2026-05
```

**只生成看板**：
```bash
/claude pilates-analyzer --action dashboard
```

## 参数说明

| 参数 | 缩写 | 必须 | 说明 |
|------|------|------|------|
| `--action` | `-a` | 否 | 执行类型：`import`/`analyze`/`dashboard`/`full` (默认) |
| `--month` | `-m` | 否 | 分析月份，格式 YYYY-MM，留空则分析所有数据 |
| `--data-path` | `-d` | 导入时必填 | Excel数据文件所在目录 |

## 输出内容

### 分析指标
- 总预约记录数
- 总会员数
- 高价值会员数（RFM Top 5%）
- 流失高风险会员数

### 会员分层（5类）
| 分层 | 判定标准 | 运营策略 |
|------|---------|---------|
| 🔥 高价值 | 7天内到店 + 月均8节以上 | VIP服务、留存优先 |
| 🟢 活跃 | 30天内到店 + 月均4-8节 | 常规维护、课程推荐 |
| 🆕 新会员 | 30天内到店 + 消费3节以内 | 欢迎引导、培养习惯 |
| 💤 沉睡 | 30-90天未到店 | 召回活动、优惠刺激 |
| ❌ 流失 | 90天以上未到店 | 流失回访、重新激活 |

### 流失风险
- 🔴 高风险：60天以上未到店
- 🟡 中风险：30-60天未到店
- 🟢 低风险：30天内有到店

### 看板图表
1. 📈 月度耗课量趋势
2. 🏪 各门店数据对比
3. 👥 会员分层分布卡片
4. ⚠️ 流失风险三色预警
5. ⭐ Top 10 高价值会员榜单
6. 📊 RFM评分分布柱状图

## 数据格式要求

Excel文件需要包含以下列（按列索引识别）：
- 第3列：会员姓名
- 第5列：上课时间
- 第6列：课程名称
- 第8列：教练姓名
- 第15列：会员卡名称

文件名需要包含门店标识：`新世纪` / `美乐城` / `鑫港` / `拉德芳斯`
