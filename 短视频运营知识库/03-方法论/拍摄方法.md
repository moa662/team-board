# 🎥 拍摄方法

---

## 📷 镜头语言与状态要求

### 眼神自然流转
*   正常讲话时眼神要自然左右扫动，不要固定盯着提示器，增加真实感。
*   **老师原话**："你一看就是在看提示器，正常人讲话我们一定会左右扫一扫。"

### 多遍拍摄法则
*   建议拍摄至少3-4遍，第三遍、第四遍的状态通常会优于前两遍。
*   **核心逻辑**：第1、2遍是机械感，第3、4遍才是"口语感"。

### 情绪掺入
*   讲案例故事时需要掺入个人情绪，避免语音语调全程没有变化。
*   **老师原话**："你少了一点起伏，语音语调稍微掺一点点自己的情绪。"

## 💡 布光技巧

### 最省钱的布光方案

1. **自然光（最好用！）**
   - 面对窗户坐，让光打在脸上
   - 不要背对窗户（脸会黑）
   - 不要侧对窗户（一半亮一半黑）

2. **补光灯方案**
   - 主灯：前面45度，比脸高一点
   - 辅灯：另一边，亮度低一点，弱化阴影
   - 背景灯：打亮背景，人物更突出

### ✅ 布光检查清单
- [ ] 脸上有没有明显的阴影？
- [ ] 眼镜有没有反光？
- [ ] 背景是不是比脸暗？
- [ ] 眼神有没有光？（眼睛里要有高光点）

### ❌ 绝对不能做
- 头顶只有一盏顶灯（像恐怖片）
- 背后有强光（逆光黑脸）
- 光线颜色太杂（冷光暖光混一起）

## 🎤 收音注意事项

> 声音比画面重要100倍！画面糊一点可以忍，声音听不清直接划走。

### ✅ 收音优先级

1. **有线领夹麦** > 无线领夹麦 > 手机自带麦克风
2. 麦的位置：嘴巴正前方10-20cm
3. 别在衣服里面，不要摩擦到衣服

### 🎧 拍摄前必查
- [ ] 有没有录进去声音？先试录5秒听一下
- [ ] 有没有杂音（空调、窗外车声、电流声）？
- [ ] 有没有喷麦？（P/B音喷到麦上）
- [ ] 声音大小是不是合适？不要太小也不要爆音

### 💡 小技巧
- 说话前先停顿0.5秒，给后期留剪辑空间
- 说错了不要停，直接重新说一遍，后期剪
- 环境太吵就等一等，或者换个地方
- 一定要戴耳机听回放！不要相信手机喇叭

## ❌ 常见拍摄错误

| 错误 | 后果 | 怎么改 |
|------|------|--------|
| **死盯着提示器** | 一看就是在读稿子，缺乏真实感 | 像日常说话一样扫视，遵循"自然视线流转" |
| 背景太乱 | 观众注意力被分散 | 找一面白墙，或者用背景布 |
| 人太小 | 看不清表情和动作 | 镜头拉近一点，人物占画面1/2 |
| 脸太黑 | 看不清表情 | 面对光源，或者开补光灯 |
| 镜头晃 | 看着晕 | 放三脚架/靠在墙上/用防抖 |
| 拍摄状态没有起伏 | 语音语调太平铺直叙 | 讲案例时故意掺入个人情绪 |

## 📸 每次拍摄前检查清单

- [ ] 手机有没有放横？（16:9还是9:16先想清楚）
- [ ] 电量够不够？别拍一半关机
- [ ] 内存够不够？别拍一半存不下
- [ ] 有没有开美颜？别开太大，假
- [ ] 背景有没有不该入镜的东西？
- [ ] 要不要换个取景场景？（咖啡厅/露台/门店）
- [ ] 提醒自己：眼神要自然流转，不要死盯提示器！
